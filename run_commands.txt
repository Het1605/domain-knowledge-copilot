# Domain Knowledge Co-Pilot - Startup Instructions

To run this project, you must start both the Backend (FastAPI) and the Frontend (Streamlit) servers.

First, open your terminal and navigate to the project root directory:
cd "/Users/het/Desktop/Domain Knowledge Co-Pilot"

---------------------------------------------------------
1. How to run the Backend (FastAPI API Server)
---------------------------------------------------------
Run the following command in your terminal:

./.venv/bin/python backend/run.py

(This starts the backend on http://localhost:8000)

---------------------------------------------------------
2. How to run the Frontend (Streamlit UI App)
---------------------------------------------------------
Open a new terminal tab/window, navigate to the project root, and run:

./.venv/bin/streamlit run frontend/app.py

(This starts the frontend on http://localhost:8501)
---------------------------------------------------------
